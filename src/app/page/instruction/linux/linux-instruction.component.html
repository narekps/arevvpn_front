Linux
1
Скачайте Happ

Скачайте актуальный релиз Happ для Linux со страницы релизов на GitHub. Доступны форматы .deb, .rpm и .pkg.

Скачать Happ для Linux (.Deb) Скачать Happ для Linux (.Rpm) Скачать Happ для Linux (.Pkg)
2
Установите приложение

Выберите способ в зависимости от вашего дистрибутива:

Debian / Ubuntu (.deb)

sudo dpkg -i Happ.linux.x64.deb

Fedora / openSUSE (.rpm)

sudo rpm -i Happ-x.x.x.x86_64.rpm

Arch Linux / Manjaro (.pkg)

sudo pacman -U Hhapp-x.x.x-x86_64.pkg.tar.zst
3
Откройте Happ и добавьте подписку
  1. Запустите Happ из меню приложений или командой happ в терминале.
  2. Нажмите кнопку «+» или перейдите в раздел «Подписки» (Subscriptions).
  3. Вставьте вашу ссылку в поле «URL» и нажмите «Сохранить».
Ваша ссылка выглядит так: {{environment.API_URL}}/key/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
4
Выберите сервер
  1. После загрузки подписки в списке появятся серверы.
  2. Выберите нужный сервер из списка.
5
Подключитесь к VPN
  1. Нажмите кнопку «Подключить» (Connect).
  2. Система может запросить пароль суперпользователя (sudo) для настройки сетевого интерфейса — введите его и подтвердите.
  3. Когда статус сменится на «Подключено» — VPN активен! ✅
