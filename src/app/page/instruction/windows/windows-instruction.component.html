Windows
1
Скачайте и установите Happ

Скачайте установочный файл Happ с официального сайта или GitHub.

Скачать Happ для Windows
  1. Скачайте файл Happ-Setup-x.x.x.exe.
  2. Запустите установщик. При запросе UAC нажмите «Да» (требуются права администратора для установки VPN-драйвера).
  3. Следуйте инструкциям мастера установки и нажмите «Установить».
2
Откройте Happ
  1. После установки Happ запустится автоматически. В дальнейшем открывайте его через значок на рабочем столе или в меню «Пуск».
  2. Happ добавит иконку в системный трей (правый нижний угол экрана).
3
Добавьте подписку
  1. В главном окне нажмите кнопку «+» или перейдите в раздел «Подписки» (Subscriptions).
  2. Выберите «Добавить подписку» (Add Subscription).
  3. Вставьте ссылку, полученную после оплаты, в поле «URL».
  4. Нажмите «Сохранить» или «OK». Happ загрузит список серверов.
Ваша ссылка выглядит так: {{environment.API_URL}}/key/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
4
Выберите сервер
  1. В выпадающем списке или на главном экране выберите нужный сервер.
  2. Рекомендуем выбирать сервер с наименьшей задержкой (ping).
5
Подключитесь к VPN
  1. Нажмите большую кнопку «Подключить» (Connect).
  2. Windows может запросить разрешение брандмауэра — нажмите «Разрешить доступ».
  3. Когда кнопка станет «Отключить» (Disconnect) и иконка в трее изменится — VPN активен! ✅
